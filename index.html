Basic Controls:
W/A/S/D Move (double tap to run)
J Attack
K Block
Space Interact
ESC Pause
C Stats/Equipment
B Inventory
X Skill

Combos:
S -> A/D -> J Shuriken
K -> S -> J Poison Blade
S -> S -> J Shadow Clone
